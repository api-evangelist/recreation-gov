# Recreation.gov (recreation-gov)

The Recreation Information Database (RIDB) provides data resources to citizens, offering a single point of access to information about recreational opportunities nationwide. The RIDB represents an authoritative source of information and services for millions of visitors to federal lands, historic sites, museums, and other attractions/resources.

## APIs

### Recreation.gov RIDB API

The Recreation Information Database (RIDB) API provides programmatic access to federal recreation data including activities, facilities, campsites, recreation areas, tours, permit entrances, events, media, links, and managing organizations. Authentication is via API key.

- **Human URL:** [https://ridb.recreation.gov/landing](https://ridb.recreation.gov/landing)
- **Base URL:** `https://ridb.recreation.gov/api/v1`
